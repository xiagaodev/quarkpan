## 上架策略

1. **定价**：稀缺度越高越贵，TPT/新加坡数学可定高端，幼小衔接走量定低价
2. **封面**：每包做3-5张素材预览图，展示内容丰富度
3. **标题关键词**：品牌名（Jolly Phonics/SAP/蒙氏/TPT）+ 人群（3-6岁/幼小衔接）+ 卖点（原版/全套/可打印）
4. **发货**：夸克网盘链接，24小时自动发货
5. **上新节奏**：先上06和02（刚需+稀缺），再上01和05（高端），最后03和04（补充）
